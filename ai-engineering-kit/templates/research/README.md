# AI Research — AI Mühendislik Merkezi

> Bu dizin projenin Claude AI ajan altyapısını geliştirmek için
> araştırma, ölçüm ve iyileştirme kaynaklarını içerir.
> **AI Engineering Kit** tarafından oluşturulmuştur.

---

## İki Komut, İki Boyut

```
/ai-upgrade       → Neye sahibiz?      → Yeni skill, MCP, repo, araç ekle
/ai-metodoloji    → Nasıl çalışıyoruz? → Bağlam, hafıza, reasoning, orchestration
```

## Dosyalar

| Dosya | Komut | İçerik |
|-------|-------|--------|
| `REPO_CATALOG.md` | `/ai-upgrade` | Repo'lar, kategori, öncelik |
| `SESSION_LOG.md` | `/ai-upgrade` | Araç ekleme oturumları, skor |
| `METODOLOJI_LOG.md` | `/ai-metodoloji` | Metodoloji denetim oturumları |

## Öncelik Kuralı

```
Metodoloji < 7  →  /ai-metodoloji (sistemi düzelt)
Metodoloji ≥ 7  →  /ai-upgrade (yeni araç ekle)
İkisi ≥ 7       →  Yeni özellik geliştir
```
